# Calibration Guide

How to align QA reviewers on scoring standards before using scores for coaching.

---

## Why calibration matters

Without calibration, QA scores are opinions, not measurements. Two reviewers scoring the same interaction should be within 1 point on each dimension.

---

## Before the first scoring cycle

1. Pick 5 interactions covering a range: excellent, good, borderline, failing, edge case
2. Each reviewer scores independently
3. Compare scores dimension by dimension
4. For any dimension with a gap of more than 1 point: discuss the specific behaviour
5. Update the rubric notes with the agreed interpretation
6. Re-score until within 1 point on each dimension

---

## Common disagreements

**Tone:** Key question: did the response acknowledge the player's emotional state before moving to resolution? One sentence is sufficient.

**Player experience:** A response ending with "let me know if you need anything" is not a clear next step. "Once you send me the transaction ID I'll escalate to billing" is.

**Accuracy:** Score 1, not 0, for language that sounds like policy but is vague. Score 0 only for specific claims that contradict known policy.

---

## Share the rubric with agents

Agents should know how they are being scored. A QA system applied without transparency damages trust.
